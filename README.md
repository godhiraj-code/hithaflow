# HithaFlow 🌿
> *Hitha (Beneficial) • Flow (Rhythm) — Where intelligence meets purity of soul.*

HithaFlow is a frictionless, empathetic companion web application designed specifically to help manage Fibromyalgia symptoms, uncover behavioral patterns, and maintain a sustainable daily energy budget (pacing).

## Features ✨
- **🧠 Low-Cognitive Load Design**: Built with a "dark mode by default" interface, avoiding bright, jarring colors that trigger sensory overload. Features a dynamic layout that reduces screen clutter when high-pain flare-ups are detected.
- **🥄 Daily Energy Budget (Spoon Theory)**: Dynamically track your physical and mental energy reserves using visual spoons. Custom alerts tell you when to rest and prevent overexertion.
- **🤒 Quick Emoji Logger**: Log pain, fatigue, sleep, and stress levels instantly using intuitive micro-interactions and animations. No typing required.
- **🚨 Flare-Up Survival Kit**: Automatically displays a prioritized action plan when a high-pain day is logged to reduce panic and promote immediate self-care.
- **📊 Smart Analytics & Pattern Spotting**: Generates daily insights based on your recent trends and flags triggers using continuous background analytics.
- **🖨️ Clinical PDF Export**: Generate clean, actionable reports for your rheumatologist or doctor in one click.
- **🌐 Offline-First Architecture**: Completely private, storing your data entirely securely on your device so it works flawlessly even without internet. 

## Technology Stack 💻
- **Core**: HTML5, Vanilla JavaScript, Vanilla CSS
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Deployment**: Vercel

## Running Locally 🚀
1. Clone the repository:
   ```bash
   git clone https://github.com/godhiraj-code/hithaflow.git
   ```
2. Navigate into the directory:
   ```bash
   cd fibroflow
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the local development server:
   ```bash
   npm run dev
   ```
